Cosa è un Array?

In JavaScript, un array è una struttura dati che contiene una collezione ordinata di elementi. Gli elementi possono essere di qualsiasi tipo di dato, come numeri, stringhe, oggetti, funzioni, ecc. Un array può essere dichiarato e inizializzato in diversi modi, ad esempio:

// Dichiarazione e inizializzazione di un array vuoto

let arrayVuoto = [];


// Dichiarazione e inizializzazione di un array con elementi

let numeri = [1, 2, 3, 4, 5];


// Array di oggetti

let persone = [

{ nome: 'Mario', cognome: 'Rossi'},

{ nome: 'Luigi', cognome: 'Verdi'}

];

Gli elementi in un array sono accessibili tramite un indice numerico, dove il primo elemento ha indice 0, il secondo elemento ha indice 1 e così via. Ad esempio:

console.log(numeri[0]);

// Output: 1

// Output: banana

console.log(persone[0].nome);

// Output: Mario

Gli array in JavaScript sono flessibili e possono crescere o ridursi dinamicamente, il che significa che è possibile aggiungere, rimuovere o modificare gli elementi in qualsiasi momento.


Posizione e Lunghezza negli Array

  • Gli elementi dentro un Array hanno una posizione che parte da 0, 1, 2, ecc.
  • La lunghezza di un Array parte invece da 1, 2, ecc.

Per accedere all'ultima posizione di un Array (e dunque verificarne la lunghezza) è sufficiente digitare array [array.lenght -1]

RICORDA: Un singolo Array può contenere altri Array al suo interno.


I metodi degli Array

Metodi mutabili

JavaScript offre molti metodi integrati per manipolare gli array. Ecco alcuni dei metodi più comuni:

  • push(): Aggiunge uno o più elementi alla fine dell'array e restituisce la nuova lunghezza dell'array

    let numeri = [1, 2, 3];

    numeri.push(4);

    console.log(numeri);// Output: [1, 2, 3, 4]

  • pop(): Rimuove l'ultimo elemento dall'array e lo restituisce

    let numeri = [1, 2, 3]

    let elementoRimosso = numeri.pop();

    console.log(elementoRimosso); // Output: 3

    console.log(numeri);// Output: [1, 2]

  • shitft(): Rimuove il primo elemento dall'array e lo restituisce

    let numeri = [1, 2, 3];

    console.log(elementoRimosso);// Output: 1

    console.log(numeri);// Output: [2, 3]

  • unshift(): Aggiunge uno o più elementi all'inizio dell'array e restituisce la nuova lunghezza dell'array

    let numeri = [2, 3];

    numeri.unshift(0, 1);

    console.log(numeri);// Output [0, 1, 2, 3]

  • splice(): Aggiunge, rimuove o sostituisce elementi di un array. Questo emtodo può essere utilizzato per una vasta gamma di operazioni complesse sugli array

    let numeri = [1, 2, 3, 4, 5];

    numeri.splice(2, 1);// Rimuove un elemento a partire dall'indice 2

    console.log(numeri);// Output: [1, 2, 4, 5]

Metodi Immutabili

  • slice(): Estrae una parte di un array e restituisce una copia di quella parte. Non modifica l'array originale

  • concat(): Crea un nuovo array combinando uno o più array esistenti con altri elementi. Non modifica gli array originali

  • map(): Crea un nuovo array applicando una funzione a ciascun elemento dell'array. Non modifica l'array originale.

  • filter(): Crea un nuovo array contenente solo gli elementi che soddisfano una determinata condizione. Non modifica l'array originale.

  • reduce(): Applica una funzione ad ogni elemento dell'array, producendo un singolo valore. Non modifica l'array originale.
