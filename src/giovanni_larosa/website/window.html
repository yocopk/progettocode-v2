WINDOW

L'oggetto globale "window",in javascript rappresenta la finestra del browser in cui viene visualizzata la pagina web. L'oggetto "window" di solito viene creato automaticamente dal browser quando viene caricata una pagina web, e contiene informazioni su di essa, nonché vari metodi e proprietà per interagire con la finestra stessa. Ecco alcuni esempi di come puoi utilizzare l'oggetto "window" in Javascript:

  • Puoi utilizzare il metodo 'alert()' per visualizzare un avviso nella finestra del browser: window.alert("Ciao Mondo!")
  • Puoi utilizzare il metodo 'open()' per aprire una nuova finestra del browser:window.open("http://www.example.com", "example", "width=600,height=400");
  • Puoi utilizzare il metodo 'close()' per chiudere la finestra del browser corrente:window.close();
  • Puoi accedere a vari oggetti come il documento corrente, i navigatori, la cronologia e le intestazioni HTTP utilizzando proprietà dell'oggetto "window". Ad esempio, per accedere all'URL del documento corrente:let url = window.location.href;
  • Puoi utilizzare il metodo 'setInterval()' per eseguire una funzione ripetutamente ad un determinato intervallo di tempo:let timer = window.setInterval(myFunction, 1000);
