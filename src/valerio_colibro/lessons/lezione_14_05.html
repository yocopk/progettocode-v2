Lezione 14/05 - Ricapitolazione

  • var si usa meno di let nella gestione di una variabile perchè non consente una buona visione della variabile. Dopodichè uso let per riassegnarle un valore;
  • const si usa quando si ha un valore costante che non si intende cambiare;
  • Attraverso const e let non possono riassegnare l'array. Dunque non useremo .push;
  • Dato che noi useremo un approccio invariabile, useremo molto const anche per la sua capacità di occupare poco spazio;
  • Approfondire i metodi degli array;
  • Esercizio: data una stringa, dimmi se è palindroma;
  • Oggetti: con let posso riassegnare una variabile, invece con const non lo posso fare;
  • Inoltre per guardare dentro una variabile, basta usare nomeVariabile. (nomeVariabilepunto);
  • Per costruire un oggetto, basta usare la classe costruttrice Object ed ecco un esempio: const obj = new Object() e avremo come output obj;
  • All'interno di Object c'è il metodo statico.keys() che permette di restituire un array sotto forma di stringa;
  • Invece .values() è un metodo statico che restituisce un array sotto forma di valori;
  • Per modificare direttamente una classe, usare direttamente il nome della classe facendo class nomeClasse { message = "Testo" };
  • Un'altra funzione per le classi è constructor(){} ed è usata quando viene creata una nuova istanza della classe ;
  • Con this.nameoaltraparola abbiamo il Polimorfismo nella classe ossia il trattare gli oggetti nello stesso modo, anche se di classe diversa;
  • Poi c'è anche l'Ereditarietà di una classe, ossia estenderla col comando extends e assegnarle altre proprietà;
  • Il metodo super() è quel metodo che richiama la classe che sto estendendo, riscrivendo metodi e proprietà delle classi precedenti;
  • Il metodo saluta esiste sull'oggetto che istanza la classe;
  • Con la parola static all'interno di saluta rendo il nome della classe metodo della classe stessa
