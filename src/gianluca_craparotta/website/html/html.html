HTML (HyperText Markup Language)

HTML è il linguaggio di markup standard utilizzato per creare e progettare pagine web. Consiste in una serie di elementi, o "tag", che definiscono la struttura e il contenuto della pagina.

Elementi HTML

Gli elementi HTML sono costituiti da un tag di apertura, il contenuto e un tag di chiusura. Ad esempio, il tag <p> viene utilizzato per definire un paragrafo: 

<p>Questo è un paragrafo</p>

Struttura di base di un documento HTML

Un documento HTML di base inizia con un'etichetta <!DOCTYPE html> che dichiara la versione di HTML utilizzata, seguita da un tag <html> che contiene l'interezza del documento HTML. 

  <!DOCTYPE html>
  <html lang="it">
      <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Titolo della pagina</title>
      </head>
      <body>
          <h1>Intestazione della pagina</h1>
          <p>Questo è un paragrafo.</p>
      </body>
  </html>

Struttura di base di un elemento HTML

Ogni elemento HTML ha una struttura di base simile:

  • Tag di apertura: Indica l'inizio dell'elemento e contiene il nome dell'elemento, eventuali attributi e i valori degli attributi.
  • Contenuto: Il testo o gli altri elementi HTML contenuti all'interno dell'elemento.
  • Tag di chiusura: Indica la fine dell'elemento e contiene solo il nome dell'elemento preceduto da una barra obliqua.
